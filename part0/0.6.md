1- user access site, HTTP GET request to https://studies.cs.helsinki.fi/exampleapp/spa
2- browser get HTML, CSS, JS and JSON files from server
3- files are load in the browser, notes list and form are displayed
4- the user write a note in the input field
5- the user clicks submit button and the event handler calls the preventDefault method to prevent send data to server and cause a new GET request (reload)
6- the new note is rendered on the page
7- an HTTP POST request called new_note_spa is executed
8- the browser send the input data from user to the server
