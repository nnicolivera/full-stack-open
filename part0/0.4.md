1- user access site, HTTP GET request to https://studies.cs.helsinki.fi/exampleapp/notes
2- browser get HTML, CSS, JS and JSON files from server
3- files are load in the browser, notes list and form are displayed
4- the user write a note in the input field
5- the user clicks submit button
6- an HTTP POST request called new_note is executed
7- the browser send the input data from user to the server
8- the submit event reload the page in the browser
9- step 1,2 and 3 again and the new note is added to the list
